## Conceptos Clave

```python
# Condicional
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

# Bucle for
for i in range(5):
    print(i)

# List comprehension
cuadrados = [x**2 for x in range(10)]

# Diccionario
persona = {"nombre": "Ana", "edad": 25}
```

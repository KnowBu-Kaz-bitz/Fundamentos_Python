Al finalizar esta semana, serás capaz de:

- ✅ Comprender qué es Python y por qué es importante
- ✅ Instalar y configurar Python en tu computadora
- ✅ Escribir y ejecutar programas básicos en Python
- ✅ Trabajar con variables y tipos de datos fundamentales
- ✅ Usar operadores aritméticos, de comparación y lógicos
- ✅ Manejar entrada y salida de datos
